# PltPics
A simple script that draws picture into `matplotlib.pyplot` using bezier curves.

## Special Thanks To
kevinjycui for https://github.com/kevinjycui/DesmosBezierRenderer

## Features
### Photo to Graph
This feature can be achieved by `DrawImage.ipynb`
